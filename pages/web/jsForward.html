JS实现网页跳转

按钮式 
 < INPUT name="百度" type="button" value="GO" onClick=" location.href='http://www.baidu.com' ">
链接式 
 < a href=" javascript:history.back() ">返回上一步< /a>
 < a href=" javascript:history.go(-1) ">返回上一步< /a>
 < a href=" <%=Request.ServerVariables("HTTP_REFERER")%> ">返回上一步< /a>
直接跳转式 
< script>window.location.href=' http://http://www.baidu.com ';< /script>
开新窗口 
< a href="javascript:" onClick=" window.open( 'http://www.baidu.com', '',
'height=500,width=611,scrollbars=yes,status=yes' )">百度< /a>
